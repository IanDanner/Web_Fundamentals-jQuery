.click/.hide/.show

Clicking the .hide button will hide this paragraph, while the .show button shows it.

.toggle

Clicking the .toggle button shows/hides this paragraph on alternating clicks.

.slideDown/.slideUp

Clicking the .slideUp button hides the paragraph in an upward direction, while the .slideDown shows the paragraph in a downward direction.

.slideToggle

Clicking the .slideToggle will hide/show this paragraph in an upward/downward direction on alternating clicks.

.fadeIn/.fadeOut

Clicking the .fadeOut button causes this paragraph to fade out, while the .fadeIn button causes it to fade in.

.addClass

Clicking the .addClass button causes this paragraph to have another class changing its color to blue.

.before

Clicking the .before button causes text to appear before this paragraph.

.after

Clicing the .after button causes text to appear after this paragraph.

.append

Clicking the .append button will added text to the current paragraph(elememt).

.html

Clicking the .html button will replace this text with new text.

.attr

Clicking the .attr button will add a new atribute to this paragraph, aligning it left.

.val

Clicking the .val button will save the current selected option (console.log it in this case).

.text

Clicking the .text button will cause all text in this selected div to be output.

.data

Clicking the .data button will store the data in the selected element.
